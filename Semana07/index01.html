PRÁCTICA CON KENYI Y SKARLY

Ejercicio 1

Crear un sitio web con HTML y CSS que incluya una página de inicio, una página de contacto y una página de servicios. La página de inicio debe tener un encabezado, un párrafo de bienvenida y una imagen. La página de contacto debe incluir un formulario con campos para el nombre, correo electrónico y mensaje. La página de servicios debe listar los servicios ofrecidos por la empresa.

HTML

El código HTML se utiliza para estructurar el contenido de la página web. Se utilizan etiquetas para definir diferentes elementos como encabezados, párrafos, imágenes y formularios. Por ejemplo, la etiqueta <h1> se utiliza para los encabezados principales, mientras que la etiqueta <p> se utiliza para los párrafos de texto.

CSS

El código CSS se utiliza para dar estilo a la página web. Se pueden definir colores, fuentes, tamaños y disposición de los elementos. Por ejemplo, se puede utilizar la propiedad color para cambiar el color del texto, o la propiedad background-color para cambiar el color de fondo de un elemento.
